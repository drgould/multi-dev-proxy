# Installation

**Homebrew (macOS/Linux):**

```sh
brew install drgould/mdp/mdp
```

**curl (macOS/Linux):**

```sh
curl -fsSL https://raw.githubusercontent.com/drgould/multi-dev-proxy/main/install.sh | sh
```

**Scoop (Windows):**

```powershell
scoop bucket add mdp https://github.com/drgould/scoop-mdp
scoop install mdp
```
